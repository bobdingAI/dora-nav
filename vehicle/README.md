# 说明

该目录为Adora机器人在DORA环境下的驱动节点

| 序号 |               文件夹名                |     功能包名称     |
| :--: | :-----------------------------------: | :----------------: |
|  1   |    adora_chassis_control_dora_node    | 机器人底盘控制节点 |
|  2   |     adora_head_control_dora_node      |   机器人头部节点   |
|  3   | adora_lifting_motor_control_dora_node | 机器人升降电机节点 |
|  4   | adora_suction_pump_control_dora_node  |     吸力泵节点     |
